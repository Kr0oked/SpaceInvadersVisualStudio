Welcome to v1.0 of my SpaceInvaders Clone.

I created this application as an educational project at the

Hochschule Ravensburg-Weingarten.



------------------------------------------------------------

----------------------START THE GAME------------------------

------------------------------------------------------------

To start the game mark the spaceinvaders file as executable

and double click it or start it from the command line.



------------------------------------------------------------

-----------------------HOW TO PLAY--------------------------

------------------------------------------------------------

Space Bar: Shot
Left 
Arrow Key: Move left

Right Arrow Key: Move right

Up Arrow Key: Special Shot

ESC: Exit Game

f: Full Screen

r: Restart Game

m: Switch Polygon Mode



------------------------------------------------------------

-----------------------COMPILE IT YOURSELF------------------

------------------------------------------------------------

On Debian:



Install the following packages:



apt-get install cmake build-essential mesa-common-dev
 
	freeglut3-dev libglew-dev libglm-dev libsoil-dev
 
	libxmu-dev libxi-dev



Execute the Build Script, output goes to the build directory

sh build.sh
